В файле README.txt описывается файловая структура проекта. 

Блок кода помечается как "0-Co" в начале каждого из файлов, где "0" это порядок файловой иерархии, а
"Co" отражает суть блока, в данном случае Co (Common) обозначает то что это общий блок кода

Кодификаторы файловой иерархии:
Матричный калькулятор - "1-MaC"
Калькулятор Обыкновенных дробей - "2-FrC"
Калькулятор Функций - "3-FuC"
Калькулятор Комбинаторики - "4-CoC"
Калькулятор Теории Вероятности и мат. статистики - "5-PrC"
Калькулятор Многочленов - "6-PoC"

После кодификатора идёт конструкция "File-__", где на месте "__" находится номер файла в конкретном блоке.
В каждом блоке нумерация начинается с 01.
Первым файлом каждого из блоков должен являться Заголовочный файл, вторым файлом является меню.

Пример:
    В блоке "1-MaC" файловая структура будет выглядеть так: 
        1-MaC-File-01-Header.h
        1-MaC-File-02-Menu.cpp
        1-MaC-File-03-*ВашКреатив*.cpp
        ...
        1-MaC-File-__-*КреативПродолжается*.cpp
        ...
